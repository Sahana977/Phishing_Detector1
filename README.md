# Phishing Detector Server

Java-based phishing detection system using VirusTotal API.

## Features
- Multi-client server using sockets
- URL phishing detection
- JSON-based communication
- Logs phishing attempts

## Tech Stack
- Java
- Sockets
- VirusTotal API
- ExecutorService

## How to Run
1. Set VIRUSTOTAL_API_KEY as environment variable
2. Run PhishingDetectorServer
3. Run PhishingDetectorClient
